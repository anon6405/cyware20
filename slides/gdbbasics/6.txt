- ${BLD}${BLK}[n]ext${RST}${RST}
  * Unlike "step", if the current source line calls a subroutine,
  this command does not enter the subroutine, but instead steps
  over the call, in effect treating it as a single source line
     
- ${BLD}${BLK}[n]ext[i]${RST}${RST}
  * steps through a single x86 instruction

 - ${BLD}${BLK}[file]${RST}${RST}
   * loads specified file into gdb
